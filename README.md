## 🧪 Instalação
```bash
# Clone este repositório
$ git clone https://github.com/JonathanFrosto/to-do-list

# Acesse a pasta do projeto no terminal/cmd
$ cd To-do-list

# Instale as dependências
$ npm install

# Execute a aplicação.
$ npm start

# Acesse http://localhost:3000 no seu navagador.
```
